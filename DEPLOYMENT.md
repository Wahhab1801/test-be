# Deploy to LiveKit Cloud

Minimal steps to ship this agent with the `lk` CLI.

## Prereqs

- LiveKit Cloud account and `lk` CLI installed (`brew install livekit`).
- `OPENAI_API_KEY` (and any other app secrets) ready.

## Deploy

```bash
lk cloud auth                           # log in
lk project list                         # optional: choose project
lk project set-default "<project>"      # optional
lk agent create --region eu-central \
  --secrets OPENAI_API_KEY="$OPENAI_API_KEY"
lk agent deploy
```

- `livekit.toml` is generated by `agent create` and used by `agent deploy`.
- The Dockerfile in this repo is used for the build.

## Monitor

```bash
lk agent logs --follow   # stream logs
lk agent list            # check status
```

Dashboard: https://cloud.livekit.io

## Update

```bash
lk agent deploy          # rebuild and roll out latest code
```

## Notes

- LiveKit injects `LIVEKIT_URL`, `LIVEKIT_API_KEY`, and `LIVEKIT_API_SECRET` automatically in cloud; you only supply app secrets like `OPENAI_API_KEY`.
- Keep the frontend `NEXT_PUBLIC_AGENT_NAME` / `LIVEKIT_AGENT_NAME` in sync with the agent you create.
